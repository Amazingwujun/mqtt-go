# Project MQTT-GO
`Mqtt-GO` 基于 [MQTT v3.1.1](http://docs.oasis-open.org/mqtt/mqtt/v3.1.1/os/mqtt-v3.1.1-os.html) 协议，采用 `go` 语言开发。

**mqtt-go** 区别与 [MQTTX](https://github.com/Amazingwujun/mqttx), mqtt-go feature 如下:

1. 支持 mqtt v3.1.1 协议
2. 不支持消息持久化
3. 不引入第三依赖
4. 不支持 tls

以上，**mqtt-go** 将会提供极简的 broker
> 构建完成的执行文件大小不会超过 ***3MB***
